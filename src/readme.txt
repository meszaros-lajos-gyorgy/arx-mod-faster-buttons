---------------------------------------------
 Faster buttons mod for Arx Fatalis
---------------------------------------------

An Arx Fatalis mod that reduces wait time between button presses

--------------
 Installation
--------------

Unpack the contents of this zip and add it into your Arx Fatalis/Arx Libertatis game's root folder.

Using Fredlllll's Arx Libertatis mod manager is recommended for easy install and uninstall:
https://github.com/fredlllll/ArxLibertatisModManager
